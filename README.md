# Platform Support
As cool as it would be to have 100% bug free code, it's sadly not a reality. As such I've written up a table containing what Zinput *does and does not* support as of today.

<center>

|                       | **Windows** | **MacOS** | **X11** | **Wayland** |
|:---------------------:|:-----------:|:---------:|:-------:|:-----------:|
| **Mouse Control**     | ✅          | ✅          | ✅          | 🏃          |
| **Mouse Listener**    | 🏃          | 🏃          | 🏃          | 🏃          |
| **Keyboard Control**  | 🏃          | 🏃          | 🏃          | 🏃          |
| **Keyboard Listener** | 🏃          | 🏃          | 🏃          | 🏃          |

</center>
